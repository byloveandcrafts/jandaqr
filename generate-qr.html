How to deploy on GitHub Pages:
1. Create a GitHub repo named wedding-seat-finder
2. Upload index.html to the repo
3. Go to Settings → Pages → Deploy from branch → main → / (root)
4. Your URL will be: https://yourusername.github.io/wedding-seat-finder/
5. Paste it above and click Generate!
